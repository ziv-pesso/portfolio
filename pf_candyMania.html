Candy Tournament

Trophies for each tournament

In this feature players try to compete for a higher score in candy themed tournaments. We decided to assign a color to each tournament so that the trophies, backgrounds and UI elements all connect to each other, creating a visual language as a result. For example the Jelly Bean Tournament has a trophy with a green icon, ribbons and background color as well.

This is a very quick and rough flow mockup that I created in order to communicate the idea with the rest of the team. The UI and flow for these screens were created by Kiki Ho.

Map Screen

UI by Priscillia Pun

Treasure Opening Screen

In this mode players try to survive multiple levels to reach the top of the hill. I was in charge of the background for this new feature. I designed it with the UI that Priscillia Pun created in mind, making sure that it would read well and avoiding tangents or other design issues. I took some of the puzzle pieces used in the game as motifs for the pillars and then added some of the shape language from the map onto the treasure chest.
